## About the Assignment
This assignment is question 4 of the given assignments. Instructions to play the game:
1. It is a number guessing game.
2. The computer has generated a random number between 1 and 100. The player has to guess this number.
3. There are 10 valid attempts to play.
4. An attempt is invalid if a value other than a number between 1 and 100 is entered.
